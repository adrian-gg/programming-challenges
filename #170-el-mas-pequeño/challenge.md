# Reto #170: El más pequeño

#### Dificultad: easy

## Enunciado

Crea una función que dada una string de numeros te devuelva el número más pequeño que se pueda formar.

No puedes usar el metodo sort o funcionalidades similares.

Ejemplo 1:
Input: "321"
Output: "123"

Ejemplo 2:
Input: "934"
Output: "349"
