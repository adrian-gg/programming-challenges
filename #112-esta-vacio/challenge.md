# Reto #112: ¿Está vacío?

#### Dificultad: Facil

## Enunciado

Dado un objeto o un array, devuelve si está vacío.

- Un objeto vacío no contiene pares clave-valor.
- Un array vacío no contiene elementos.

Ejemplo 1:

Input: obj = `{ x: 5, y: 42 }`
Output: false

Ejemplo 2:

Input: obj = `{}`
Output: true

Ejemplo 3:

Input: obj = `[null, false, 0]`
Output: false
